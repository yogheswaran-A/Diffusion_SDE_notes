Ito's (Di)Lemma

Or how to differentiate a function of a stochastic process.

Differentiability

Probably one of the most fundamental uses of calculus is the derivation of functions. A function ff is differentiable if the following value f′(x)f′(x) exists: f′(x)=limh→0f(x+h)−f(x)h(1)(1)f′(x)=limh→0f(x+h)−f(x)h If we let hh go towards zero, a function is called differentiable, if the fraction converges towards some constant value.

Let’s look at an example of how this might work. We’ll need an additional mathematical trick called L’Hopitals rule which says that for evaluating the limit of a fraction we can simply derive both nominator and denominator with respect to the same variable and still obtain the valid result. Applying L’Hopitals rule often simplifies the computation of the derivative since we’re always working with the limit of a fraction.

Let’s try to compute the derivative of the squared function f(x)=x2f(x)=x2: f′(x)=limh→0f(x+h)−f(x)h=limh→0(x+h)2−x2h=limh→0x2+2hx+h2−x2h=limh→0∂∂h 2hx+h2∂∂hh=limh→02x+2h1=2x⇐Applying L'Hopitals rule⇐Evaluating h to zero(2)(3)(4)(5)(6)(7)(2)f′(x)=limh→0f(x+h)−f(x)h(3)=limh→0(x+h)2−x2h(4)=limh→0x2+2hx+h2−x2h(5)=limh→0∂∂h 2hx+h2∂∂hh⇐Applying L'Hopitals rule(6)=limh→02x+2h1⇐Evaluating h to zero(7)=2x Sure enough it’s the correct result which we anticipated. In effect, we’re zooming infinitely far into the function and ask ourselves how the function changes in this tiny window hh. Visually, this looks something like this for the exponential function f(x)=exf(x)=ex:

What this functions tells us is that we can approximate an arbitrarily complex, differentiable function with a linear function for a extremely small window limh→0limh→0. But in order for the function to be differentiable, the limit has to actually converge to a linear function as we decrease the window size hh.

Unfortunately, for stochastic processes this is not as straight forward and we will require some more elaborate tools to show some notion of differentiability.

Stochastic Processes

In order to keep things simple in the following steps, we will work with the Brownian Motion.
Brownian motion WtWt is defined through the following properties:

  • W0=0W0=0
  • Independent increments: covariance C[Wt+u−Ws,Ws]=0C[Wt+u−Ws,Ws]=0 for u≥0u≥0 and s≤ts≤t
  • Gaussian increments: Wt+u−Wt∼N(0,u)Wt+u−Wt∼N(0,u)
  • Continuous paths in time tt.

Now it turns out that there exists a stochastic differential equation which fulfills all of the properties above. This SDE in question is dxt=dWt=ϵdt−−√;ϵ∼N(0,1)(8)(8)dxt=dWt=ϵdt;ϵ∼N(0,1) Intuitively, we equate the infinitesimal change in xtxt with Brownian Motion which in turn is defined as the standard normally distributed random variable ϵϵ scaled by dt−−√dt. The problem of classical differentiability of stochastic processes lies precisely in this SDE as we defined the change with respect to dtdt. By defining the infinitesimal change dtdt we are acknowledging that we could always use a shorter dtdt and zoom even further into the time axis. After all, the infinitesimal change of the Brownian Motion dWtdWt is defined as a limit in time not unlike the limit we used to show the differentiability of the quadratic function: dWt=limΔt→0Wt+Δt−Wt∼N(0,Δt)(9)(9)dWt=limΔt→0Wt+Δt−Wt∼N(0,Δt) While ΔtΔt goes rapidly towards zero, it will actually never be exactly zero. Thus, if we were to zoom into the time axis we would realize that the Brownian Motion keeps moving randomly for whatever time resolution we choose. In turns out that Brownian Motion actually has fractal properties which are probably the trippiest mathematical animations you can experience without doing actual acid.

No matter how far we zoom into the Brownian Motion, we will always encounter a Brownian Motion on a finer time scale since dxtdxt moves randomly on any time scale we choose. Probably the best animation for that is directly from the Wikipedia page of Brownian Motion:

You might wonder: Well, why is that a problem with respect to classical differentiability? For that we can simply evaluate the differential limΔt→0Wt+Δt−WtΔt(10)(10)limΔt→0Wt+Δt−WtΔt but alas, WtWt is by definition a Normally distributed random variable. So let’s have a look at the mean and variance of the differential operator: limΔt→0E[Wt+Δt−WtΔt]=limΔt1ΔtE[Wt+Δt−Wt∼N(0,Δt)]=limΔt1Δt0=0(11)(12)(13)(11)limΔt→0E[Wt+Δt−WtΔt]=limΔt1ΔtE[Wt+Δt−Wt⏟∼N(0,Δt)](12)=limΔt1Δt0(13)=0 and limΔt→0V[Wt+Δt−WtΔt]=limΔt→01Δt2V[Wt+Δt−Wt∼N(0,Δt)]=limΔt→01Δt2Δt=limΔt→01Δt=∞(14)(15)(16)(17)(14)limΔt→0V[Wt+Δt−WtΔt]=limΔt→01Δt2V[Wt+Δt−Wt⏟∼N(0,Δt)](15)=limΔt→01Δt2Δt(16)=limΔt→01Δt(17)=∞ Both the mean and the variance are possibly the worst values you can expect in terms of functional analysis. The mean is zero, indicating we have no derivative what so ever while the variance goes to infinity which is equally unusable. Ultimately, we can’t derive a Wiener process in the classical sense.

It turns out that Kiyosi Ito had a series of great insights that we can use. But before we can dive into his ideas, we first have to learn about Taylor expansions …

Taylor Expansion

The Taylor expansions or Taylor series is one of the most ubiquitous mathematical tools in applied math. Once at the DeepBayes summer school in Moscow, a fellow attendee and physicist said that if you have no clue what to do next with your equations, do a Taylor expansion and see if it gets you ahead.

The core idea of a Taylor expansion is to approximate a function locally around a root point with a series of terms which rely on the derivatives of the function. So for example a function might be a polynomial of order 10 but locally, we only need a quadratic function to approximate it quite well.

Mathematically, a Taylor expansion of a infinitely differentiable function f(x)f(x) around a root point x0x0 is defined as f(x)|x0=∑∞n=0f(n)(x0)n!(x−x0)nf(x)|x0=∑n=0∞f(n)(x0)n!(x−x0)n In order to keep things simple we will only work with a Taylor expansion of the second order, meaning that we will stop the sum after the term with the second order derivative. Practically, many problems are posed as linear or quadratic problems so the need seldomly arises to compute higher order Taylor expansions (at least in machine learning where computing higher order gradients at scale can be expensive).

So we’ll be working with the following sum: f(x)|x0≈f(x0)+f′(x0)1!(x−x0)+f′′(x0)2!(x−x0)2=f(x0)+f′(x0)(x−x0)Δx+12!f′′(x0)(x−x0)2Δx2=f(x0)+f′(x0) Δxlinear in Δx+12f′′(x0) Δx2quadratic in Δx(18)(19)(20)(18)f(x)|x0≈f(x0)+f′(x0)1!(x−x0)+f″(x0)2!(x−x0)2(19)=f(x0)+f′(x0)(x−x0)⏟Δx+12!f″(x0)(x−x0)2⏟Δx2(20)=f(x0)+f′(x0) Δx⏟linear in Δx+12f″(x0) Δx2⏟quadratic in Δx where Δx=(x−x0)Δx=(x−x0) signifies the distance of x to the root point x0x0. By using the second order Taylor expansion we approximate the higher order polynomial f(x)f(x) with just its first and second order derivative packed into a polynomial in ΔxΔx. The locality of the Taylor expansion around the x0x0 is essential for the approximation since we use the first order order derivative f′(x0)f′(x0) and second order derivative f′′(x0)f″(x0) explicitly evaluated at x0x0.

  • The second order term of the Taylor approximation plays a more significant role and we can see that it tries to approximate the function around x0x0 with a quadratic function.
  • Furthermore the sum of the two terms approximate the original function around the root point more precisely than either could have done on its own.

Now the question can be raised on how this could be applied to stochastic processes …

Ito’s Lemma

Let’s assume we a classic SDE with a drift term μ(t,Xt)μ(t,Xt) and a diffusion term σ(t,Xt)σ(t,Xt) which together form: dXt=μ(t,Xt)dt+σ(t,Xt)dWt(21)(21)dXt=μ(t,Xt)dt+σ(t,Xt)dWt and dWtdWt is the infinitesimal differential of a Wiener process WtWt. Such a process is commonly called an Ito drift-diffusion process.

Now let’s say that we have some function f(t,Xt)f(t,Xt) that takes whatever value XtXt is at the moment tt and returns some other value YtYt such that we have Yt=f(t,Xt)(22)(22)Yt=f(t,Xt) We could use relatively easy functions such as as the exponential function eXteXt or the quadratic function X2tXt2 for starters. In the financial markets, these functions ff quickly get very complex as stock prices are routinely modeled as stochastic differential equations with ff capturing complex relationships like a portfolio performance or default probability.

Since we are working with infinitesimal differentials we would like to know how YtYt changes for very small time differentials dtdt. So we actually want to be able to define the following equation: dYt=df(t,Xt)(23)(23)dYt=df(t,Xt) In order to answer that question, Ito’s lemma applies a Taylor expansion to f(t,Xt)f(t,Xt) with special numerical conditions for the infinitesimal values. There are a few constraints on ff, though. It has to be twice differentiable with respect to XtXt and at least once differentiable with respect to tt. If these equations are met we can start deriving!

The first step is to define the Taylor expansion for f(t,Xt)f(t,Xt) in it’s general form around the root point (t0,X0)(t0,X0): f(t,Xt)≈f(t0,X0)+∂f(t0,X0)∂t(t−t0)Δt+∂f(t0,X0)∂Xt(Xt−X0)ΔXt+12∂2f(t0,X0)∂Xt(Xt−X0)2ΔX2t=f(t0,X0)+∂f(t0,X0)∂tΔt+∂f(t0,X0)∂XtΔXt+12∂2f(t0,X0)∂XtΔX2t(24)(25)(24)f(t,Xt)≈f(t0,X0)+∂f(t0,X0)∂t(t−t0)⏟Δt+∂f(t0,X0)∂Xt(Xt−X0)⏟ΔXt+12∂2f(t0,X0)∂Xt(Xt−X0)2⏟ΔXt2(25)=f(t0,X0)+∂f(t0,X0)∂tΔt+∂f(t0,X0)∂XtΔXt+12∂2f(t0,X0)∂XtΔXt2

The next step is to examine the Taylor expansion in the limit lim[t→t0,Xt→X0]lim[t→t0,Xt→X0]. This is of interest as we are again interested in the infinitesimal behavior of f(t,Xt)f(t,Xt), namely df(t,Xt)df(t,Xt): By pulling the root evaluation f(t0,X0)f(t0,X0) over to the left side we lay the groundwork for the differential. limt→t0,Xt→X0f(t,Xt)−f(t0,X0)=limt→t0,Xt→X0∂f(t0,X0)∂tΔt+∂f(t0,X0)∂XtΔXt+12∂2f(t0,X0)∂XtΔX2t(26)(26)limt→t0,Xt→X0f(t,Xt)−f(t0,X0)=limt→t0,Xt→X0∂f(t0,X0)∂tΔt+∂f(t0,X0)∂XtΔXt+12∂2f(t0,X0)∂XtΔXt2

In fact the limit above is precisely the differentiability operator from the very beginning. Remember that we define the differentiability as the difference of two evaluations for an ever more decreasing difference in their arguments. This is precisely what we are defining in the limit above by moving tt ever closer to t0t0 and simultaneously XtXt towards X0X0. Furthermore the limit also allows us to rewrite the difference ΔtΔt and ΔXtΔXt in their infinitesimal differential form dtdt and dXtdXt.

So we obtain the following: limt→t0,Xt→X0f(t,Xt)−f(t0,X0)=limt→t0,Xt→X0∂f(t0,X0)∂tΔt+∂f(t0,X0)∂XtΔXt+12∂2f(t0,X0)∂XtΔX2t(27)(27)limt→t0,Xt→X0f(t,Xt)−f(t0,X0)=limt→t0,Xt→X0∂f(t0,X0)∂tΔt+∂f(t0,X0)∂XtΔXt+12∂2f(t0,X0)∂XtΔXt2

where the left sides simplifies to the differential of the function ff: df(t,Xt)=∂f(t0,X0)∂tdt+∂f(t0,X0)∂XtdXt+12∂2f(t0,X0)∂XtdX2t(28)(28)df(t,Xt)=∂f(t0,X0)∂tdt+∂f(t0,X0)∂XtdXt+12∂2f(t0,X0)∂XtdXt2

The next step is substituting dXt=μ(t,Xt)dt+σ(t,Xt)dWtdXt=μ(t,Xt)dt+σ(t,Xt)dWt into the equation: df(t,Xt)=∂f(t0,X0)∂tdt+∂f(t0,X0)∂XtdXt+12∂2f(t0,X0)∂XtdX2t=∂f(t0,X0)∂tdt+∂f(t0,X0)∂Xt(μ(t,Xt)dt+σ(t,Xt)dWt)+12∂2f(t0,X0)∂Xt(μ(t,Xt)dt+σ(t,Xt)dWt)2=∂f(t0,X0)∂tdt+∂f(t0,X0)∂Xt(μ(t,Xt)dt+σ(t,Xt)dWt)+12∂2f(t0,X0)∂Xt(μ(t,Xt)2dt2+2μ(t,Xt)σ(t,Xt)2dtdWt+σ(t,Xt)2dW2t)(29)(30)(31)(32)(29)df(t,Xt)=∂f(t0,X0)∂tdt+∂f(t0,X0)∂XtdXt+12∂2f(t0,X0)∂XtdXt2(30)=∂f(t0,X0)∂tdt+∂f(t0,X0)∂Xt(μ(t,Xt)dt+σ(t,Xt)dWt)+12∂2f(t0,X0)∂Xt(μ(t,Xt)dt+σ(t,Xt)dWt)2(31)=∂f(t0,X0)∂tdt+∂f(t0,X0)∂Xt(μ(t,Xt)dt+σ(t,Xt)dWt)(32)+12∂2f(t0,X0)∂Xt(μ(t,Xt)2dt2+2μ(t,Xt)σ(t,Xt)2dtdWt+σ(t,Xt)2dWt2)

Now comes a pivotal part in the derivation in which examine how dtdt and dWtdWt behave when multiplied or squared. The differential Wiener process can be rewritten as dWt=ϵdt−−√dWt=ϵdt. Thus we have the following time-dependent terms appearing in the equation above: dt2dt2, dtdWt=ϵdt1.5dtdWt=ϵdt1.5 and dW2t=ϵ2dt=dtdWt2=ϵ2dt=dt under the mean-square interpretation which states E[ϵ2]=V[ϵ]=1E[ϵ2]=V[ϵ]=1 for ϵ∼N(0,1)ϵ∼N(0,1).

The important aspect of simplifying Ito’s lemma is to think about how dt2dt2, dt1.5dt1.5 and dtdt behave for infinitesimal changes. Any dtkdtk with k>1k>1 and dt<1dt<1 will decrease by an order of magnitude faster to zero than dtdt itself for the infinitely small values that we’re dealing with. So if we evaluate for a infinitesimal small dtdt, the terms dt2dt2 and dt1.5dt1.5 will be smaller by larger order of magnitudes. This allows us to simply drop them from our equation.

So ultimately, it turns out that the derivative of the function f(t,X)f(t,X) with an Ito drift-diffusion process as input is an Ito drift-diffusion process itself. Albeit with a few first and second order derivatives sprinkled in between.

Thus we can model the function f(t,Xt)f(t,Xt) just like any other drift-diffusion process and can evaluate the distribution of such a process at a later point in time.
